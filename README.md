Exploratory Data Analysis on School Dataset

Project Overview

Objective

To analyze real-world messy data, clean it, and extract meaningful insights about student performance.

Dataset Description

The dataset contains the following features:

Math Scores
Language Scores
Parent Literacy Rate
Other demographic or school-related attributes

Data Cleaning Steps

The dataset was preprocessed to ensure accuracy and reliability:

* Filled missing values using **mean (average) imputation**
* Removed **duplicate records**
* Verified dataset consistency after cleaning

Key Insights

1. Most Students Score Around Average

* The distribution of math scores shows that most students fall in the **mid-range**
* Extreme high and low scores are less common

2. Parent Literacy Impacts Language Scores

* A positive relationship exists between **parent literacy and student language performance**
* Students with literate parents tend to score higher

3. Consistent Performance Across Subjects

* Math and language scores are **positively correlated**
* Students performing well in one subject often do well in others

 4. Data Quality is Reliable

* Missing values and duplicates have been handled
* The dataset is now clean and consistent

5. Not All Factors Have Equal Impact

* Heatmap analysis shows:

  * **Strong correlations** → significant relationships
  * **Weak correlations** → minimal influence

6. Dataset is Suitable for Further Analysis

* Clean and balanced data enables:

  * Predictive modeling
  * Trend analysis


Conclusion

Student performance is moderately distributed, and factors such as **parent literacy have a meaningful positive influence** on academic outcomes.
